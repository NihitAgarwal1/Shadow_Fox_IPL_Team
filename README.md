Kolkata Knight Riders — IPL Team Website

A responsive fan website dedicated to the **Kolkata Knight Riders (KKR)**, 
one of the most celebrated franchises in the Indian Premier League (IPL). 
Built and deployed with ⚡ Vercel.

🌐 Live Demo
[kkr-ipl-team.vercel.app](https://kkr-ipl-team.vercel.app)

## 📖 About
This project is a frontend web application showcasing the Kolkata Knight Riders 
IPL cricket team — including team information, squad details,and more. It is 
designed to give fans a sleek and engaging experience.

 Features
-Team overview and franchise history
-Player profiles and squad listing
-Fully responsive design for mobile and desktop

 Tech Stack
- Frontend:HTML / CSS / JavaScript  *(or React — update as needed)*
- Deployment: [Vercel](https://vercel.com)
